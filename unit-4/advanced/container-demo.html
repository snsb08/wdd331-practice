Using min-width: 0 on Flex Items

Some of the most frustrating CSS bugs are caused not by rules you wrote, but by rules the browser assumed. Two defaults in particular catch developers off guard repeatedly: the min-width: auto that every flex item inherits, and the box-sizing: content-box that every element starts with. Neither is a bug. Both made sense when CSS was designed. But both consistently produce layout behavior that looks broken until you know the rule behind it. https://byui-cse.github.io/wdd331r-course/unit/4/learn/choose/flex-min-width-zero.html
