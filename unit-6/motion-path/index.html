Motion Path

The Offset Properties

  • offset-path: Defines the path the element follows.
    • SVG path string
    • circle()
    • ellipse()
    • url()
  • offset-distance:from 0% (start) to 100% (end).
  • offset-rotate: Controls the element's rotation as it travels.
    • auto
    • deg
  • offset-anchor: Which point on the element is pinned to the path.
  • offset: Shorthand for all four properties above.
  • Use transform-only motion for simple arcs, bounces, and throws where approximate curves are fine and the element does not need to rotate along the path.
  • Use offset-path when the path changes direction, when the element must face forward, when the path was designed in a vector tool and needs to be followed precisely, or when you want to animate along a circle or ellipse without trigonometry.
